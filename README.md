# Digital_Ordering_In_Restaurant
 The site provides a  solution by  replacing physical menus with a digital platform, allowing customers to place orders directly from their  own devices. This system ensures that orders reach the kitchen instantly, reducing service delays and  minimizing the need for additional waitstaff.
